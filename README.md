# Task 1: Iris Flower Classification – CodSoft Internship

## Objective:
Build a machine learning model to classify iris flowers into Setosa, Versicolor, and Virginica based on petal and sepal measurements.

## Tools Used:
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn (Random Forest Classifier)

## Accuracy Achieved:
Achieved over **95%** accuracy using Random Forest.
